# OSI 7 Layer 모델

### 정의

- 네트워크 프로토콜과 통신을 7계층으로 표현

### 목적

- 프로토콜을 기능별로 나누고 계층별로 구분
- 벤더간 호환성을 위한 표준 필요 -> 쉬운 접근성으로 기술의 발전

### 역사

- 1970년대 초 네트워크는 정부 또는 특정 벤더에서 독점 개발, 공개형 모델 필요
- 1970년대 말 ISO(국제표준화기구)에 의해 관리
- 1984년 ISO 7498 릴리즈
